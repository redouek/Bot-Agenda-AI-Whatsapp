Obrigatorio

1. Chave Gemini API

Ela permite que o bot interprete mensagens em linguagem natural.

Passo a passo
  1. Acesse Google AI Studio - API keys.
  2. Entre com sua conta Google.
  3. Clique em Create API key.
  4. Copie a chave que comeca com AIza e cole no onboarding.

Obrigatorio

2. Google Calendar OAuth

Ele permite que voce autorize o bot a criar e consultar eventos na sua agenda.

Passo a passo
  1. Acesse Google Cloud - Credentials.
  2. Crie ou selecione um projeto.
  3. Ative a Google Calendar API.
  4. Configure a OAuth consent screen como External para teste.
  5. Crie uma credencial OAuth 2.0 do tipo Web application.
  6. Em Authorized redirect URIs, use http://localhost:3000/oauth/callback.
  7. Copie Client ID e Client Secret para o onboarding.

Opcional

3. Football Data API

Use apenas se quiser consultas sobre futebol. O bot de agenda funciona sem isso.

Passo a passo
  1. Acesse football-data.org/client/register.
  2. Crie uma conta gratuita.
  3. Copie a API key recebida e cole em Extras.

Depois do teste local

Oracle Cloud

Como voce ainda nao tem conta nem servidor configurado, a ordem certa e validar tudo localmente primeiro. Depois fazemos a etapa Oracle: criar conta Always Free, VM, dominio/IP, variaveis, processo permanente e URL OAuth nova.
