Tutorial de instalação — passo a passo

O que você vai fazer

  1. Instalar o Docker Desktop
  2. Baixar o projeto

1

Instalar o Docker Desktop

O Docker é o programa que vai rodar o bot no seu computador. Você não precisa entender como ele funciona — só instalar.

  1. Acesse o site oficial: docker.com/products/docker-desktop
  2. Clique em Download Docker Desktop e escolha a versão para o seu sistema (Windows ou Mac).
  3. Abra o instalador baixado e siga as instruções na tela.
  4. Após instalar, abra o Docker Desktop. Ele vai aparecer na barra de tarefas (Windows) ou na barra de menus (Mac) com um ícone de baleia.
Pronto quando você ver a mensagem "Docker Desktop is running" na tela inicial do app.
O Docker precisa ficar aberto em segundo plano enquanto o bot estiver rodando. Você não precisa mexer nele depois de abrir.
2

Baixar o projeto

Você tem duas opções para baixar o projeto:

Opção A — Baixar como ZIP (mais fácil)
  1. Acesse a página do projeto no GitHub.
  2. Clique no botão verde Code → Download ZIP.
  3. Extraia o arquivo ZIP em uma pasta de fácil acesso (ex: sua área de trabalho).
Opção B — Via terminal (para quem já usa Git)
git clone https://github.com/redouek/Bot-Agenda-AI-Whatsapp.git
Anote onde você salvou a pasta do projeto — você vai precisar abrir o terminal nela no Passo 5.
3

Criar a chave da Gemini API

O Gemini é a inteligência artificial do Google que interpreta suas mensagens. A chave é gratuita.

  1. Acesse: aistudio.google.com/app/apikey
  2. Faça login com sua conta Google.
  3. Clique em Create API key.
  4. Escolha um projeto Google Cloud (pode criar um novo).
  5. Copie a chave gerada — ela começa com AIza...
Guarde essa chave em local seguro. Não compartilhe com ninguém — ela dá acesso à sua conta da API.
Pronto quando você tiver uma chave no formato AIzaSy...
4

Criar as credenciais do Google Calendar

Isso permite que o bot leia e crie eventos na sua agenda. São necessárias duas etapas: ativar o serviço e criar a credencial.

4.1 — Criar um projeto no Google Cloud
  1. Acesse: console.cloud.google.com
  2. No topo da página, clique no seletor de projetos → Novo projeto.
  3. Dê um nome qualquer (ex: "whatsapp-bot") e clique em Criar.
4.2 — Ativar a Google Calendar API
  1. No menu lateral, acesse APIs & Services → Library.
  2. Pesquise por "Google Calendar API".
  3. Clique nela e depois em Enable.
4.3 — Configurar a tela de consentimento OAuth
  1. Acesse APIs & Services → OAuth consent screen.
  2. Escolha External e clique em Create.
  3. Preencha o nome do app (ex: "Meu Bot") e seu e-mail de suporte.
  4. Clique em Save and Continue nas próximas telas até chegar em Back to Dashboard.
4.4 — Criar as credenciais OAuth
  1. Acesse APIs & Services → Credentials.
  2. Clique em + Create Credentials → OAuth 2.0 Client IDs.
  3. Em Application type, escolha Web application.
  4. Em Authorized redirect URIs, clique em Add URI e cole exatamente:
    http://localhost:3000/oauth/callback
  5. Clique em Create.
  6. Copie o Client ID e o Client Secret exibidos.
Pronto quando você tiver o Client ID (termina em .apps.googleusercontent.com) e o Client Secret (começa com GOCSPX-).
5

Rodar o bot

Agora você vai abrir o terminal na pasta do projeto e rodar um único comando.

Como abrir o terminal na pasta certa
  1. Windows: Abra a pasta do projeto no Explorador de Arquivos → clique na barra de endereços → digite cmd → Enter.
  2. Mac: Clique com botão direito na pasta do projeto → Serviços → Novo Terminal na Pasta.
  3. VS Code: Abra a pasta no VS Code → menu Terminal → New Terminal.

Com o terminal aberto na pasta do projeto, cole o comando:

docker compose up --build

Aguarde. Na primeira vez vai baixar as dependências — pode demorar 2 a 5 minutos dependendo da sua internet.

Pronto quando aparecer no terminal:
[server] Painel disponível em http://localhost:3000
Deixe essa janela do terminal aberta enquanto usar o bot. Fechar o terminal encerra o bot.
6

Configurar pelo painel

Abra o navegador e acesse: http://localhost:3000

Você verá um formulário. Preencha cada campo com as informações que você coletou nos passos anteriores:

Número WhatsApp (self-chat ID)

É o seu próprio número, no formato: DDI + DDD + número + @c.us

Exemplos:

5511999999999@c.us  ← celular brasileiro (55 + DDD + número)
5521988888888@c.us
Não coloque o + na frente. O número não pode ter traços ou espaços.
Chave da API Gemini

Cole a chave que você copiou no Passo 3 (começa com AIza...).

OAuth Client ID e Client Secret

Cole os valores copiados no Passo 4.4.

ID do Calendário

Use o e-mail da sua conta Google (ex: seunome@gmail.com) ou simplesmente primary.

Minutos de lembrete

Quantos minutos antes de um evento você quer receber o lembrete. O padrão é 15.

Clique em Salvar configuração.

Depois clique em Conectar Google Calendar. Uma janela do Google vai abrir — faça login e clique em Permitir.

Pronto quando aparecer "Google Calendar conectado!" no painel.
7

Conectar o WhatsApp

O painel vai exibir um QR code. Você precisa escanear com o seu celular.

  1. No celular, abra o WhatsApp.
  2. Toque nos três pontos (Android) ou em Configurações (iPhone).
  3. Toque em Dispositivos conectados.
  4. Toque em Conectar dispositivo.
  5. Aponte a câmera para o QR code na tela do computador.
Pronto quando o painel mostrar "WhatsApp conectado".
O QR code expira a cada ~20 segundos e é renovado automaticamente. Se passar da hora, só aguardar o novo aparecer.

Dúvidas frequentes

O bot para quando fecho o computador?

Sim. Para deixar o bot rodando 24h, você precisaria de um servidor (VPS). Em casa, o bot funciona enquanto o computador estiver ligado e o Docker aberto.

Preciso escanear o QR code toda vez?

Não. A sessão do WhatsApp fica salva. Você só escaneia na primeira vez (ou se o WhatsApp desconectar por inatividade).

Como faço para parar o bot?

No terminal onde está rodando, pressione Ctrl + C. Para reiniciar depois: docker compose up (sem o --build).

Minha conta do WhatsApp pode ser banida?

O bot usa o WhatsApp Web de forma automatizada, o que tecnicamente viola os termos de uso do WhatsApp. Para uso pessoal o risco é baixo, mas existe. Use por sua conta e risco.

O bot vai custar algo?

Provavelmente não. A API Gemini tem um nível gratuito generoso para uso pessoal. O Google Calendar API também é gratuito. O único custo seria de infraestrutura se você quiser rodar em um servidor.
